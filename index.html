Volume Knob Kit Configurator

Pounamu Electronics Limited

This tool utilises the WebUSB API and has been tested using Google Chrome. The WebUSB API generally works on Chromium based browsers. Please consult the compatability list
Windows Users

Windows users need to configure the USB driver for this device correctly before using the config tool for updating the Volume Knob Kit and writing custom configurations

  1. Install Zadig
  2. Open Zadig and select Options > List All Devices
  3. Press and hold the Update button on the Volume Knob Kit while plugging the kit into your PC via USB. Keep the Update button held throughout the entire process
  4. While keeping the Update button held, select the new USB device listed from the device dropdown menu in Zadig (generally named USB Module)
  5. Select WinUSB (any version number) as the new driver and click "Install Driver", "Replace Driver" or "Reinstall Driver"
  6. Wait for the driver installation to complete. Once completed, you may let go of the Update button
Linux Users
  1. Download the required rule file and move it to /etc/udev/rules.d/
  2. Run the following commands:
    sudo udevadm control --reload-rules
    sudo udevadm trigger
Firmware Updating
  1. Download the latest firmware .bin file
  2. Select the firmware file (.bin file)
  3. Hold the update button on the volume knob kit while connecting it to your PC via USB
  4. Release the update button on the volume knob kit when the RED power light is on
  5. Click the "Update Device" button and select the "Unknown device from Qinheng Electronics"
Kit Configuration
Any Volume Knob Kit shipped before July 1st 2026 requires a firmware update to v4.0.0 or higher

  1. Select your headunit brand/type
  2. Now we can configure the different functions
    0 0 0 0 0
  3. Hold the volume knob down while connecting the volume knob kit to your PC via USB. Keep the knob held down
  4. Release the volume knob when the white status light shines
  5. Windows users, you may need to use the Zadig tool again to assign the WinUSB driver to this device. Follow the Windows Users instructions (above) if it is not
  6. Click the "Write Config" button and select RE_SWC from the device list
  7. Your configuration will be written to the volume knob kit and the device will auto-reboot
