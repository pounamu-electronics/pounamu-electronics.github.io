Volume Knob Kit Updater

1. Select the firmware file (.bin file)

2. Hold the update button on the volume knob kit while connecting it to your PC using a USB C data and power cable

3. Release the update button on the volume knob kit when the RED power light is on

4. Click the "Update Device" button and select the "Unknown device from Qinheng Electronics"

Your volume knob kit has been updated!
